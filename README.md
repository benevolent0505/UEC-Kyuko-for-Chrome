UEC 休講
====

電気通信大学の[休講情報](http://kyoumu.office.uec.ac.jp/kyuukou/kyuukou.html)を見れるChrome Appです。

## 概要

[電通大の休講情報ページをいい感じに見るChromeアプリを作った(あとDentoo.LT #12に参加した)](http://benevolent0505.hatenablog.com/entry/2015/10/28/164400)
